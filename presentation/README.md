# DIP Presentation — Slidev

Presentation on the **Dependency Inversion Principle** for *Software Engineering and Principles*.

## Run

```bash
cd presentation
npm install
npm run dev          # opens http://localhost:3030
```

## Build / export

```bash
npm run build        # static SPA in dist/
npm run export       # PDF (requires playwright-chromium: npm i -D playwright-chromium)
```

## Editing

All content lives in `slides.md`. Each `---` block is a slide.
Theme: `seriph` · highlighter: `shiki`.
